# **App Name**: CYBIM

## Core Features:

- Play Campaigns: Plays the selected campaign in a fullscreen infinite loop with ultra-fast transitions.
- Add Campaign: Adds a new campaign with an auto-generated sequential name. Opens the campaign editor.
- Campaign Editor: Allows adding images and videos, reordering items, and deleting items from a campaign.
- Image Duration Selection: Sets the display duration for images within a campaign (5s, 10s, 15s, 20s, 30s, 60s). Videos use original duration.
- Screen Orientation: Sets the screen orientation to Landscape, Reverse Landscape, Portrait, or Reverse Portrait. Applied instantly and persists.
- Auto Start: Toggles auto-start functionality. If enabled, the app immediately plays the last campaign on launch and after device reboot.
- Storage: Stores campaigns locally. Supports USB media selection. Operates offline without internet dependency.

## Style Guidelines:

- Primary color: #05feff
- Background: #101316
- Text color: #FFFFFF
- Font optimized for readability on TV screens.
- Design optimized for Android TV remote (DPAD navigation, large focusable items, no touch-based UI).
- Clean, modular layout with clearly defined sections for campaign management and settings.
- Simple, clear icons to represent different actions and settings within the app.
